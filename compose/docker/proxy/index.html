Installation Instructions:

macOS:

sudo security add-trusted-cert -d -r trustRoot -k /Library/Keychains/System.keychain orodc-proxy-ca.crt

Linux (Debian/Ubuntu):

sudo cp orodc-proxy-ca.crt /usr/local/share/ca-certificates/
sudo update-ca-certificates

Linux (RHEL/CentOS):

sudo cp orodc-proxy-ca.crt /etc/pki/ca-trust/source/anchors/
sudo update-ca-trust

Windows:

Double-click the certificate and install to "Trusted Root Certification Authorities"
